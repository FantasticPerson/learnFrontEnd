#### web安全色
```
在256色计算机系统上总能避免抖动的颜色
采用十六进制记法，使用值：00,33，66,99，CC,FF的三元组都认为是web安全的
```

#### 绝对长度单位
```
英寸(in)
厘米(cm)
毫米(mm)
点(pt)   印刷度量单位
派卡(pc)  印刷度量单位
```
#### 相对长度单位
```
em,ex,px
1个em定义为一种给定字体的font-size值，是跟随父元素字体大小改变的
ex是指字体中小写字母x的高度

许多用户代理的做法是 取em的值，再取其一半作为ex的值
```
